## 🚀 Sobre o Projeto
O **CoursePlatform** é uma solução robusta projetada para operar como um modelo de **Software as a Service (SaaS)**. A plataforma permite que múltiplos instrutores gerenciem suas próprias organizações, cursos e alunos em um ambiente isolado e seguro.

Este projeto foi desenvolvido focando em **escalabilidade**, **código limpo** e **experiência do usuário (UX)**.

---

## 🛠️ Destaques Técnicos & Funcionalidades

### **Arquitetura & Backend (Python/Django)**
* **Multi-Tenancy:** Isolamento lógico de dados para garantir que cada organização acesse apenas suas informações.
* **Django Signals:** Notificações em tempo real para engajamento entre usuários.
* **Dashboard Customizado:** Painéis completos para Instrutores e Alunos, sem depender do admin padrão do Django.
* **Autenticação Segura:** Registro de usuários com aprovação manual e recuperação de senha.

### **Frontend & Interface (Tailwind CSS)**
* **UX Moderna:** Interface responsiva e customizada com **Tailwind CSS** e **Alpine.js**.
* **Engajamento:** Sistema de Q&A com votos e fórum para discussões entre membros.
* **Templates Modulares:** Organização baseada em partials para alta manutenibilidade.

---

## 💻 Tecnologias Utilizadas
* **Linguagem:** Python
* **Framework:** Django (ORM, Signals, Templates)
* **Banco de Dados:** MySQL
* **Interface:** Tailwind CSS e Alpine.js
* **Versionamento:** Git com metodologia **GitFlow**
